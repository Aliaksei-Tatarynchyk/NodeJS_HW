Home Page

You have to login first in either of way to see the content:

  • POST: /api/auth - body should be JSON with Login/Password pair, e.g. { "username":"login", "password": "password" }
  • GET: Google
  • GET: Facebook
  • GET: Twitter

After this you will be able to call following endpoints using generated authentication token passed along with "x-access-token" header:

  • GET|POST: /api/products
  • GET: /api/products/:id
  • GET: /api/products/:id/reviews
  • GET: /api/users
